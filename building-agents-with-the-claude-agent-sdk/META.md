## 摘要

本文介绍了 Claude Agent SDK（前身为 Claude Code SDK）的设计理念和使用方式。核心设计原则是**给 Agent 一台电脑**——通过 bash、文件系统、代码生成、MCP 等工具，让 Claude 像人类程序员一样工作。文章围绕 Agent 的三步循环（收集上下文 → 执行操作 → 验证结果）展开，并分享了多种实用模式。
