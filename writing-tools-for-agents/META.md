## 摘要

本文分享了 Anthropic 内部通过迭代优化工具设计的方法论。核心洞察：**工具是确定性系统与非确定性 Agent 之间的契约**，设计方式与传统 API 完全不同。文章提供了一个三阶段开发流程（原型 → 评估 → 与 Agent 协作优化），以及 5 大工具设计原则（选对工具、命名空间、有意义的返回值、Token 效率、优化工具描述）。
