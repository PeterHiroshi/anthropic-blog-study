## 摘要

本文介绍了 Agent Skills 框架——一种通过结构化目录（包含 `SKILL.md`）向 Agent 注入专业知识、工作流程和可复用能力的机制。Skills 支持自动发现（Claude 判断相关性后自动加载）和手动调用（`/skill-name`），并遵循**渐进式信息披露**原则最小化上下文消耗。同时，Agent Skills 已成为跨平台开放标准，被 Cursor、GitHub、VS Code 等十余个工具采纳。
