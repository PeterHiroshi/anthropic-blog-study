## 摘要

本文来自 Anthropic 与数十家企业合作构建 LLM Agent 系统的实践总结。核心观点是：**最成功的 Agent 系统往往不是最复杂的，而是最简单且适合需求的**。文章定义了 Workflow 与 Agent 的区别，介绍了 5 种核心 Workflow 模式，并给出了完整的 Agent 设计原则和工具设计指南。
