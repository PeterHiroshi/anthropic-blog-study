## 摘要

本文详细介绍了 Anthropic Research 功能背后的多 Agent 系统架构与提示工程实践。系统采用主 Agent（Claude Opus 4）协调多个子 Agent（Claude Sonnet 4）并行研究，相比单 Agent 性能提升 90.2%。文章总结了 8 条提示工程原则，并讨论了 token 效率、评估方法和生产挑战。
